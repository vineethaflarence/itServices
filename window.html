Setting up 802.1X for Wireless on Windows 7/8/10

Contributed By: Nehal J Wani

Installing the CA Cert

Download the CA certificate file from: http://radius.iiit.ac.in/IIIT-H_RADIUS_CA.der

Select "Open With" and click on "OK"

Once the download finishes, click on "Open"

The certificate is not trusted. Click on "Install Certificate"

The "Certificate Import Wizard" will open up. Click on "Next".

Select "Place all certificates in the following store" and then click on "Browse".

Select "Trusted Root Certificate Authorities", then click on "OK".

The certificate store should now point to "Trusted Root Certificate Authorities". Click on "Next".

Click on "Finish".

A security warning will be displayed. Match the fingerprint with the one shown below and then click on "Yes".

A dialog will be displayed stating "The import was successful". Click on "OK".

Now, if you open the DER file again, it should show that the certificate is trusted.

Connecting to WiFi/First time Auth

Connect to the Wireless SSID: wifi@iiith

Enter your 802.1X Credentials.

Windows will throw another security alert. Don't worry, server room doesn't share the private key with anyone else, neither are we using a globally known CA. So, nobody can create a rogue server with the same CA as us. Click on connect.

Done!

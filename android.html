Setting up 8021.X for Wireless on Smart Phone (Android)

Contributed By: Nehal J Wani

Installing the Radius CA Certificate

Method 1

Visit http://radius.iiit.ac.in and click on IIIT-H_RADIUS_CA.pem

Assign some name to this Certificate and make sure you remember it. Make sure you select Wifi in the second drop down. Then click on OK.

Method 2

Follow this only if Method 1 doesn't work for you.

First, transfer the certificate: http://radius.iiit.ac.in/IIIT-H_RADIUS_CA.pem using some mechanism to the Android Phone. Then, navigate to Settings->Security

Click on "Install From Storage"

Then select "Internal Storage"

Then navigate to the location where you kept the downloaded certificate.

Select the previously downloaded CA certificate file.

Assign some name to this Certificate and make sure you remember it. Make sure you select Wifi in the second drop down. Then click on OK.

Android may warn you that we may monitor your network. But that is something that the sysadmins are not going to do, since this certificate is only required for transacting with the radius server, and your connections to the internet/intranet will not be using the certificate. So you can simple ignore this warning.

Connecting to WiFi

Connect to the Wireless SSID: wifi@iiith

Make sure the EAP method is PEAP. If Phase2 auth is none, then the server automatically sets it for you. Otherwise you can select "MSCHAPv2". In the CA Certificate field, select the certificate which you installed above.

Scroll down, enter you 802.1X credentials then click OK and you are good to go!

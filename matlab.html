MATLAB

Matlab Installation (Windows) License Expiry Date 11-09-2024

  1. Copy the Matlab folder from Run –> \matlab.iiit.ac.in (for Windows OS ); username: matlab ; password: matlab at 123; OR Alt+F2–> smb://matlab.iiit.ac.in . (for Linux OS) (Anonymous; no password)
  2. Copy the Matlab folder which contains license.dat & key to the local machine.
  3. Launch the MATLAB Installer.
  4. Choose the ‘offline’ or ‘Use a File Installation Key’ installation method.
  5. Enter your File Installation Key (FIK), when asked, and proceed through the installation.
  6. Once your installation completes, the activation client will automatically launch
  7. In the activation client, select “Activate manually without the Internet”
  8. Select the first option, “Enter the full path to your license file…”
  9. Click browser, and navigate to your license file. MATLAB will save a copy of the license file in a safe place.
  10. If you receive an ‘Activation Complete’ message, MATLAB is installed and activated.

Matlab Installation (Linux)

Download, extract and run ./matlab from bin folder http://matlab.iiit.ac.in/R2019b.tar

NOTE: This is a Network license and will work only within the campus. Those who need to have a standalone version (that will work outside the campus as well) may write to help at iiit.ac.in with a short justification for the need.

Matlab Activation/Renewal via MathWorks

  1. Start the MATLAB application
  2. On the Home tab, in the Resources section, select Help > Licensing > Activate Software.
  3. Choose Whether to Activate Automatically or Manually
  5. Log in to Your MathWorks Account
  6. Select the License
  7. Confirm Your Activation Information
  8. Complete the Activation
